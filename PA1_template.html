Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc. Create strategy for filing in all of the missing values with the mean for that interval.

getAvgSteps <- function(interval) 
         { 
          AvgInterval[AvgInterval$Interval == interval, ]$Steps
}

Create a new dataset that is equal to the original dataset but with the missing data filled in.

activityUpd <- readActivity
count <- 0  
for (i in 1:nrow(activityUpd)) 
    {
     if (is.na(activityUpd[i, ]$steps)) {
        activityUpd[i, ]$steps <- getAvgSteps(activityUpd[i, ]$interval)
        count <- count + 1
    }
}
cat("Total ", count, "NA values were filled.")
